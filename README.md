# MIT Mysore Timetable System

## Quick Start

### 1. Start Server
```bash
START_SERVER.bat
```

### 2. Open Browser
```
index.htm
```

### 3. Use System
- Login → Add Subjects → Add Faculty → Generate Timetable

---

## Requirements
- Python 3.7+
- Flask, Flask-CORS, Supabase

```bash
pip install flask flask-cors supabase
```
